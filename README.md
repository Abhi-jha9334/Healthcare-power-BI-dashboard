# 🏥 Healthcare Power BI Dashboard

> An interactive Healthcare Analytics Dashboard built using Microsoft Power BI to analyze patient demographics, disease prevalence, and health risk factors through dynamic data visualizations and actionable insights.

---

## 📖 Project Overview

The *Healthcare Power BI Dashboard* is a Business Intelligence project developed to transform raw healthcare data into meaningful insights. The dashboard enables users to analyze patient demographics, identify disease prevalence, evaluate health risks, and monitor healthcare performance using interactive reports.

This project demonstrates practical skills in *Power BI, Power Query, DAX, Data Modeling, and Data Visualization*.

---

## 🎯 Project Objectives

- Analyze patient demographics and healthcare trends.
- Monitor key healthcare performance indicators (KPIs).
- Identify disease prevalence and health risk factors.
- Provide interactive visualizations for better decision-making.
- Build a clean and professional Business Intelligence dashboard.
